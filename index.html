Welcome to Chatbot API

This API is hosted on Vercel as a Python serverless function.

Use the following endpoint for testing:

POST /api/chatbot

Send JSON like:

{"message": "Hello AI"}

Example with curl:

curl -X POST https://chatbot-api-pi-swart.vercel.app/api/chatbot -H "Content-Type: application/json" -d '{"message": "Hello AI"}'
